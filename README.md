# ProficiencyPro | AI-Age Skills & Pivot Tracker

ProficiencyPro is a modern, responsive, dark-themed web application interface designed to help professionals navigate career transitions in the era of artificial intelligence. It features automated risk analysis dashboards, interactive test clusters, tier-based monetization paths, gated content vaults, and an integrated floating AI assistant framework.

---

## 🚀 Features

*   **AI Career Shift Analyzer:** An interactive interface with dynamic animations designed to capture user roles and output simulated career assessments.
*   **Quick-Select Skill Clusters:** Interactive tag selectors that allow instantaneous user testing for key modern professions (Design, Writing, Finance, Tech).
*   **Dynamic Analytics Dashboard:** Split-card architecture categorizing core skill competencies into *High-Risk* or *Future-Proof Premium* brackets.
*   **Monetization Infrastructure:** A sleek 3-tier responsive pricing matrix layout complete with custom badges, call-to-actions, and interactive item configurations.
*   **Gated Content System:** A structural content locker equipped with blur filters (`backdrop-filter`) and CSS-pulsing animation properties to incentivize platform conversion.
*   **Floating Chat Framework:** A beautifully styled sliding customer support chatbot element complete with asynchronous messaging spaces, input rules, and multi-device adjustments.
